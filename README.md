# DeFi Pro

A modular DeFi project with core modules: lending, DEX, yield, stablecoin, and governance.

## Quick Start
- Backend: Node.js HTTP server at port 3000.
- Frontend: Static page in folder frontend/public.
- Tests: npm test using Node test runner.

## Modules
- Lending: simple interest example.
- DEX: constant product swap output.
- Yield: naive auto-compounding.
- Stablecoin: overcollateralized mint calculator.
- Governance: minimal proposal and voting.

## Security
This repository is a learning scaffold, not production-ready.
